# PingPongAI

Frontend - React
Backend - Laravel, express
Database - PostgreSQL
Server - nginx

Запуск проекта:
git clone https://github.com/RadicalIguana/PingPongAI.git
git checkout dev

Laravel

cp .env.example .env
composer install 
php artisan serve

React & express

cd react
npm install
npm run dev
node server

PostgreSQL
docker-compose up
